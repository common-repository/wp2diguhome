== Description ==

同步发表 WordPress 博客日志到 嘀咕,初次安装必须设置后才能使用。欢迎到 [liqiu.info球哥的地盘](http://www.liqiu.info/) ,本插件地址： [wp2diguhome](http://www.liqiu.info/2010/12/wp2diguhome/) 。

Version 1.0.1 加入功能
1。支持可选择的自定义短地址提供商，暂时只支持3个服务商

2。支持将标签发送到嘀咕

3。支持自定义发送短地址还是原地址

Version 1.0 支持功能

1。支持将Wordpress中文章标题发布到嘀咕

2。支持将Wordpress中文章地址转换为短地址发布到嘀咕


Version 1.0 不支持功能

1。不支持将Wordpress中文章内容发布到嘀咕

2。不支持将Wordpress中文章原地址发布到嘀咕

2。不支持同步文章分类自定义

== Installation ==

具体情况请查看插件页面 [wp2diguhome](http://www.liqiu.info/2010/12/wp2diguhome/) ，如有疑问请回复。

1. 1.0.1版，上传 `wp2diguhome.php`,'readme.txt', 到 `/wp-content/plugins/wp2diguhome` 目录

2. 在Wordpress后台控制面板"插件(Plugins)"菜单下激活wp2diguhome插件

3. 在Wordpress后台控制面板"配置(Settings)->wp2diguhome"菜单下设置插件的必须信息。（只有经过设置，插件才能正常使用）

== Frequently Asked Questions ==

= 为什么我装了插件会报该插件的Warning错误 =

*您也可以在插件页面 [wp2diguhome](http://www.liqiu.info/2010/12/wp2diguhome/) ，提出自己遇到的问题

*经本人测试由Godaddy提供的免费空间会报这样的错误，建议使用收费空间，如若购买Godaddy主机，请选择美国机房，亚太机房的空间在国内访问有点问题。来个购买链接，[传送门](http://www.godaddy.com/hosting/hosting.aspx?ci=22124&isc=IAPtno138)

= 安装启用插件后并没有同步到嘀咕? =

*在Wordpress后台控制面板"配置(Settings)->wp2diguhome"菜单下设置插件的必须信息。
